# RB Telemetry Coach Privacy Disclosure

## Submission purpose

This is the short store-form privacy and data disclosure artifact for copy/paste into browser extension submission forms. The long-form public privacy policy remains in `docs/privacy-policy.md`, and host access wording remains in `docs/permission-justification.md`.

RB Telemetry Coach does not collect or transmit user data.

The extension reads Garage 61 lap comparison and telemetry data already loaded in the active browser page. This data is processed locally in the browser to generate coaching findings. No telemetry, lap metadata, driver names, URLs, analytics, error logs, or usage data is sent to the developer or to any third-party service.

The extension does not use remote servers, analytics, crash reporting, advertising, or third-party APIs. It does not persistently store telemetry or analysis reports.
