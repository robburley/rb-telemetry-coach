# Chrome Web Store Listing Description

RB Telemetry Coach adds a focused coaching panel to Garage 61 analysis pages, helping sim racers compare two selected laps and understand where time is being gained or lost.

When you open a Garage 61 lap analysis with exactly two active laps, the extension reads the telemetry already loaded by that page and generates deterministic coaching feedback for the currently selected distance slice. Findings can highlight differences in braking, throttle use, steering, speed, gear, RPM, and racing line where the required telemetry is available.

RB Telemetry Coach is designed for practical lap review:

* Compare the two laps currently selected in Garage 61
* Analyze the active zoomed distance range instead of the whole lap
* Get concise, evidence-based coaching findings
* Review telemetry differences directly inside Garage 61
* Keep analysis local in your browser

The extension only runs on Garage 61 pages. It does not collect, transmit, sell, store, or share telemetry data, lap metadata, driver names, URLs, analytics, error logs, or usage data. It does not use remote servers, AI APIs, analytics, crash reporting, advertising, or third-party services.

RB Telemetry Coach is an independent extension and is not affiliated with, endorsed by, sponsored by, or officially connected to Garage 61.
